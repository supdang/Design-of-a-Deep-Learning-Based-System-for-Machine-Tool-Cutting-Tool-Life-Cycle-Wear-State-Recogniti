# 基于深度学习的机床刀具生命周期磨损状态识别与诊断系统

## 项目简介

本项目是一个基于深度学习的机床刀具生命周期磨损状态识别与诊断系统，采用LSTM（长短期记忆）网络对刀具磨损状态进行智能识别。系统能够处理多传感器数据，实现端到端的刀具磨损状态诊断，为智能制造提供可靠的刀具健康管理解决方案。

## 主要特性

- 🔬 **智能诊断**：基于LSTM深度学习模型，准确识别刀具磨损状态
- 📊 **多传感器融合**：支持切削力、振动、声发射等多源传感器数据
- 🎨 **可视化界面**：提供直观的GUI界面，实时显示诊断结果
- ⚡ **高效处理**：优化的算法实现，支持实时诊断
- 🔧 **模块化设计**：易于扩展和维护的系统架构
- 📈 **完整流程**：涵盖数据处理、模型训练、预测推理全流程

## 核心功能

### 1. 数据处理
- **数据加载**：支持MAT、CSV等多种数据格式
- **信号去噪**：基于小波变换的自适应去噪算法
- **特征工程**：滑动窗口技术构建时间序列样本
- **数据增强**：添加噪声、幅度缩放等增强方法

### 2. LSTM模型
- **网络结构**：2-3层LSTM，支持自定义参数
- **训练优化**：Adam优化器，学习率调度，早停机制
- **模型管理**：支持模型保存、加载和版本管理
- **性能评估**：准确率、精确率、召回率、F1分数等指标

### 3. GUI界面
- **数据可视化**：原始信号、频谱分析、统计特征展示
- **训练监控**：实时显示训练损失和准确率曲线
- **结果分析**：混淆矩阵、ROC曲线、性能指标展示
- **用户交互**：直观的参数配置和操作流程

### 4. 系统功能
- **项目管理**：支持新建、打开、保存项目
- **批量处理**：支持大批量数据的快速处理
- **结果导出**：支持图表和数据结果的导出
- **日志记录**：完整的操作日志和错误追踪

## 安装与使用

### 环境要求
- **操作系统**：Windows 10/11、Linux、macOS
- **Python版本**：3.11+
- **内存要求**：8GB RAM以上
- **显卡要求**：支持CUDA的NVIDIA显卡（可选）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd tool-wear-diagnosis-system
   ```

2. **创建虚拟环境**
   ```bash
   # 使用conda
   conda create -n tool_wear python=3.11
   conda activate tool_wear
   
   # 或使用venv
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **生成演示数据（可选）**
   ```bash
   python create_demo_data.py
   ```

### 运行系统

```bash
# 启动系统
python main.py

# 调试模式
python main.py --debug

# 设置日志级别
python main.py --log-level DEBUG
```

### 使用指南

1. **数据加载**
   - 点击"浏览..."选择数据文件
   - 点击"加载数据"按钮
   - 查看数据可视化结果

2. **模型配置**
   - 在左侧面板设置模型参数
   - 调整隐藏层大小、LSTM层数等

3. **模型训练**
   - 设置训练参数（批量大小、训练轮数等）
   - 点击"开始训练"按钮
   - 监控训练进度和性能指标

4. **模型预测**
   - 训练完成后点击"开始预测"
   - 查看预测结果和置信度
   - 分析混淆矩阵和性能指标

5. **结果分析**
   - 在"结果分析"标签页查看详细结果
   - 导出图表和数据

### 模块说明

#### config.py
系统配置管理，包含所有配置参数和常量定义。

#### data_loader.py
数据加载器，支持多种数据格式的加载和解析。

#### preprocessor.py
数据预处理器，实现信号去噪、归一化、时间序列切分等功能。

LSTM模型实现，包含模型定义、训练、预测等功能。

GUI主窗口，实现用户界面和交互逻辑。

#### logger.py
日志系统，提供统一的日志记录功能。
